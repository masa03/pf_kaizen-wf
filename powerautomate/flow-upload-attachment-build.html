添付ファイルアップロードフロー（2-7）構築手順

タスク: 2-7 ／ フロー名: 改善提案_添付ファイルアップロード

用途: Power Apps申請フォームから添付ファイルをドキュメントライブラリにアップロード

各式の「コピー」ボタンをクリックすると、改行なしでクリップボードにコピーされます。

フロー概要

[Power Apps (V2) トリガー]
    │
    ├── 入力: RequestID (テキスト)
    ├── 入力: FileName (テキスト)
    ├── 入力: FileContent (テキスト)
    ├── 入力: FileCategory (テキスト)  ← [v10] 追加
    │
    ▼
[SharePoint - ファイルの作成]
    │  サイト: 改善提案システム
    │  ライブラリ: 添付ファイル (AttachmentFiles)
    │  ファイル名: FileName
    │  ファイルコンテンツ: FileContent
    │
    ▼
[SharePoint - ファイルのプロパティの更新]
       サイト: 改善提案システム
       ライブラリ: 添付ファイル (AttachmentFiles)
       ID: [ファイルの作成].ItemId
       RequestID: RequestID（入力パラメータ）
       ファイル種別 Value: FileCategory（入力パラメータ）  ← [v10] 追加

Step 1: フロー作成

  1. Power Automate → マイフロー → 新しいフロー → インスタントクラウドフロー
  2. フロー名: 改善提案_添付ファイルアップロード
  3. トリガー: Power Apps (V2) を選択

Step 2: トリガーの入力パラメータ設定

トリガーの「入力の追加」で以下4つを追加:

パラメータ名種類必須備考
RequestIDテキストはい
FileNameテキストはい
FileContentテキストはい
FileCategoryテキストはい[v10] 改善前/改善後/その他

Step 3: アクション1 — ファイルの作成

  1. 新しいステップ → 「SharePoint」→ ファイルの作成
サイトのアドレス 対象サイトURL
フォルダーのパス
/AttachmentFiles
ファイル名 動的コンテンツから「FileName」を選択
ファイルコンテンツ（式）
dataUriToBinary(triggerBody()['text_2'])
3つのテキストパラメータは text(RequestID), text_1(FileName), text_2(FileContent) にマッピングされる。Power Appsからはdata URI文字列で渡されるため、dataUriToBinary() でバイナリに変換が必要。

Step 4: アクション2 — ファイルのプロパティの更新

  1. 新しいステップ → 「SharePoint」→ ファイルのプロパティの更新
サイトのアドレス 対象サイトURL
ライブラリ名 添付ファイル
ID 動的コンテンツから「ファイルの作成」→ ItemId を選択
リクエストID 動的コンテンツから「RequestID」を選択
ファイル種別 Value 動的コンテンツから「FileCategory」を選択 [v10]

Step 5: 保存・テスト

  1. 保存 をクリック
  2. テスト → 手動 でテスト実行（Power Appsから呼び出し前にフロー単体で確認）

フロー共有時の接続設定（実行のみのユーザー）

WriteSecurity=2 環境での予防策: このフローはインスタントフロー（Power Apps トリガー）のため、デフォルトでは実行ユーザーの権限で SharePoint API が呼ばれます。現時点では WriteSecurity 未設定のリスト（添付ファイルライブラリ）のみに書き込むため直接影響はありませんが、将来の権限変更に備えてフロー所有者の接続を使用する設定にしてください。
  1. フローの詳細ページを開く（編集画面ではなく概要ページ）
  2. 右側の 「実行のみのユーザー」 セクションの 「編集」 をクリック
  3. 「接続の使用」で SharePoint の行を確認
  4. 「この接続を使用する」（フロー所有者の接続）を選択する
  5. 「保存」 をクリック

Power Apps からの呼び出し

データソース接続

  1. Power Apps Studio → 左メニュー「データ」→ データの追加
  2. 「Power Automate」→ 「改善提案_添付ファイルアップロード」フローを選択
