# 画面レイアウト設計書

**バージョン**: v10
**作成日**: 2026-03-07
**参考**: 改善提案シート ver9.18 (PDF)

> 本ドキュメントはPower Apps各画面のレイアウト（セクション配置・サイズ比率）を定義する。
> 論理的な項目定義は `docs/spec/index.md` を参照。

### 画像表示の仕様

| 項目 | 仕様 |
|------|------|
| 表示対象 | FileCategory="改善前" / "改善後" の画像ファイル |
| 表示方式 | Power Apps Image コントロールで直接埋め込み |
| 画像URL | SharePointドキュメントライブラリのファイルURL（`gSharePointSiteUrl & "/" & ファイルパス`） |
| 複数画像 | 同一種別の画像が複数ある場合、先頭1件を表示（残りはリンク表示） |
| 画像なし | 種別の画像がない場合、「画像なし」プレースホルダーを表示 |
| アスペクト比 | `ImagePosition.Fit` で元画像の比率を保持 |

### 評価結果セクションの表示条件

| 条件 | 表示内容 |
|------|----------|
| ステータス = 承認済 / 差戻 | 評価結果セクション全体を表示 |
| ステータス = 申請中 / 課長評価中 / 部長評価中 | 評価結果セクションを非表示 |
| 表彰区分 != 改善提案 | 4軸スコア部分を非表示（判定・金額・コメントのみ表示） |
| 部長評価データなし | 部長評価カラムを非表示 |

## 4. 共通レイアウトルール

### 色定義

| 用途 | 色 |
|------|-----|
| 背景 | RGBA(245, 245, 245, 1) |
| コンテンツ背景 | RGBA(255, 255, 255, 1) |
| 読取専用フィールド背景 | RGBA(240, 240, 240, 1) |
| セパレータ | RGBA(220, 220, 220, 1) |
| ステータス: 申請中 | RGBA(0, 120, 212, 1) |
| ステータス: 課長評価中 | RGBA(255, 185, 0, 1) |
| ステータス: 部長評価中 | RGBA(255, 140, 0, 1) |
| ステータス: 承認済 | RGBA(16, 124, 16, 1) |
| ステータス: 差戻 | RGBA(196, 49, 75, 1) |
| ステータス: 取下げ | RGBA(128, 128, 128, 1) |

### フォントサイズ

| 用途 | サイズ |
|------|--------|
| 画面タイトル | 22 |
| セクションタイトル | 16 |
| 項目ラベル | 13 |
| 項目値 | 13 |
| 補足テキスト | 10-12 |
| ステータスバッジ | 12 |

### ラベル幅

| コンテキスト | ラベル幅 |
|---|---|
| 閲覧画面 項目ラベル | 120px |
| 閲覧画面 項目値 | 400px |
