## 1. システム概要

### 1.1 目的

業務改善提案の申請から評価・承認までのプロセスを電子化し、SharePoint + Power Platform 上で一元管理するシステムです。従来のエクセル管理から閲覧・編集、承認フローの効率化を図ります。

### 1.2 利用規模

| 項目           | 値                    |
| -------------- | --------------------- |
| 利用ユーザー数 | 15,000人              |
| 改善メンバー   | 申請者含め最大10名/件 |
| 想定年間件数   | 数百件                |

### 1.3 技術スタック

| コンポーネント | 技術/サービス                     | 用途                                 |
| -------------- | --------------------------------- | ------------------------------------ |
| フロントエンド | Power Apps（キャンバスアプリ）    | 申請画面、評価画面、閲覧画面         |
| データストア   | SharePoint Lists                  | 申請データ、マスタデータ、評価データ |
| ワークフロー   | Power Automate                    | メール通知、ステータス遷移           |
| ファイル管理   | SharePoint ドキュメントライブラリ | 添付ファイルの保管                   |
| 認証/ID        | Microsoft Entra ID                | ユーザー認証、社員番号との紐付け     |

### 1.4 表彰区分

本システムでは4つの表彰区分を扱います。

| 表彰区分               | 褒賞金額                   | 評価方法                                |
| ---------------------- | -------------------------- | --------------------------------------- |
| 改善提案               | 等級に応じて 300〜10,000円 | 4軸スコアリングによる通常評価フロー     |
| 小集団 パール賞（3等） | 3,000円（固定）            | 事前評価済 — スコアリング不要、承認のみ |
| 小集団 銅賞（2等）     | 5,000円（固定）            | 事前評価済 — スコアリング不要、承認のみ |
| 小集団 銀賞（1等）     | 10,000円（固定）           | 事前評価済 — スコアリング不要、承認のみ |

**改善提案（個人）の等級テーブル:**

| 等級 | 素点範囲 | 褒賞金額 |
| ---- | -------- | -------- |
| 6等  | 29点以下 | 300円    |
| 5等  | 30〜39点 | 600円    |
| 4等  | 40〜49点 | 1,000円  |
| 3等  | 50〜59点 | 3,000円  |
| 2等  | 60〜69点 | 5,000円  |
| 1等  | 70点以上 | 10,000円 |

## 2. 実装機能

| No. | 機能               | 概要                                                                                                                                                                                                                                                                             |
| --- | ------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1   | 申請資料作成・登録 | 提案内容（テーマ・問題点・改善内容）、改善分野実績（効果金額自動算出）、改善メンバー（最大10名）、添付ファイルを入力し、SharePoint Listsに登録。社員情報・組織情報・承認者はログインユーザーから自動取得                                                                         |
| 2   | 申請フロー         | 申請者が提出 → 課長評価・承認 → 褒賞金額5,000円以上の場合のみ部長評価・承認。差戻時は申請者が修正・再提出（課長評価から再開）。課長本人が申請者の場合は課長承認をスキップし部長が第1承認者                                                                                       |
| 3   | 申請閲覧・評価     | 申請内容の読み取り専用表示（プレビュー・登録後閲覧・承認時確認）。課長/部長は4軸スコアリング（効果・独創性・努力工夫・応用範囲）で評価し、職能換算を経て等級・褒賞金額を自動判定。小集団 パール賞（3等）/小集団 銅賞（2等）/小集団 銀賞（1等）の場合はスコアリング不要で承認のみ |
| 4   | 承認フロー         | 課長承認時に褒賞金額で自動分岐（<5,000円: 承認完了 / ≥5,000円: 部長承認へ）。承認・差戻に応じてステータスを自動更新し、最終褒賞金額を確定                                                                                                                                        |
| 5   | メール通知         | 承認依頼（課長/部長へ）、差戻通知（申請者へ）、承認完了（申請者+承認者へ）の3種を自動送信                                                                                                                                                                                        |

## 5. 改善分野の入力・算出・保存仕様

改善分野マスタには14分野が登録されており、**分野種別**（金額算出 / 直接入力 / テキスト）によって入力UI・効果金額の算出方法・保存内容が異なります。

**関連リスト構造:**

| リスト         | カラム                                                                                                                 |
| -------------- | ---------------------------------------------------------------------------------------------------------------------- |
| 改善分野マスタ | 分野コード / 分野名 / 単位 / 金額換算単価 / 分野種別                                                                   |
| 改善分野実績   | リクエストID / 改善分野コード / 改善分野名 / 実績値 / 仕掛り金額 / 金額換算単価 / コメント / 効果金額算出式 / 効果金額 |

### 5.2 分野種別ごとの処理比較

| 項目                     | 金額算出型（CAT-01〜05） | 金額算出型・リードタイム短縮（CAT-06）             | 直接入力型（CAT-07〜11） | テキスト型（CAT-12〜14） |
| ------------------------ | ------------------------ | -------------------------------------------------- | ------------------------ | ------------------------ |
| **申請者の入力項目**     | 実績値（数値）           | 実績値（分/月）+ 仕掛り金額（円）                  | 効果金額（円/月）        | 効果テキスト（自由記述） |
| **効果金額の算出方法**   | 実績値 × 換算単価        | 仕掛り金額がそのまま効果金額（実績値との乗算なし） | 入力値がそのまま効果金額 | 効果金額なし（0円）      |
| **効果金額合計への加算** | あり                     | あり                                               | あり                     | なし                     |
